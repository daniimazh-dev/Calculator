# Calculator <img width="26" height="26" alt="app_icon" src="https://github.com/user-attachments/assets/9b4d524e-5f02-44c7-a9e8-498bae4a4653" /> 

> A modern calculator for **Android**, built using **Jetpack Compose**.  
The project has a big emphasis on converters and design in **Material 3**.

---

## ✨ Opportunities 

- ➕ Complex multitasking mathematical operations
- 💱 25+ converters
- 🎨 Modern UI and big customization
- 💬 Commentaries on operations history
- 📈 Graphs of functions
- 💾 Compactness (15MB)
- 🧩 Open source

---

## 🛠️ Technologies used

- **Jetpack Compose (Android)**
- **Ktor (Backend)**
